# 📘 영어 단어 학습 프로그램 (학년별 · 수능 필수)

이 저장소는 **학년별 필수 영어 단어 + 수능 필수 2,000단어**를 기반으로 한
**HTML 기반 영어 단어 학습 프로그램**입니다.

설치 없이 웹에서 바로 실행되며, **회당 30단어 학습**, **난이도별(B1/B2/C1) 분리**,
**퀴즈 및 오답노트** 기능을 포함합니다.

---

## 🔎 주요 특징

* 학년별 단어 구성

  * 초등 저학년 / 고학년
  * 중1 / 중2 / 중3
  * 고1 / 고2 / 고3
  * 수능 필수(core)

* 학습 구조

  * 회당 30단어 자동 분할
  * 단계별 학습 (단어 → 예문 → 퀴즈 → 오답)
  * 난이도 선택 (B1 / B2 / C1)

* 기술 구성

  * HTML + JavaScript (순수 프론트엔드)
  * JSON 데이터 기반
  * GitHub Pages 배포

---

## 📂 프로젝트 구조

```
english-vocab
├─ index.html          # 학습기 메인 화면
├─ README.md           # 안내 문서
└─ data
   ├─ elem_low.json    # 초등 저학년
   ├─ elem_high.json   # 초등 고학년
   ├─ middle_1.json    # 중1
   ├─ middle_2.json    # 중2
   ├─ middle_3.json    # 중3
   ├─ high_1.json      # 고1
   ├─ high_2.json      # 고2
   ├─ high_3.json      # 고3
   └─ csat_core.json   # 수능 필수 2,000단어
```

---

## 📄 JSON 데이터 형식

모든 단어 데이터는 아래 형식을 따릅니다.

```json
{
  "word": "analyze",
  "pos": "verb",
  "meaning": "분석하다",
  "example": "Researchers analyze the data carefully.",
  "level": "csat_core",
  "difficulty": "B2"
}
```

* `word` : 영어 단어
* `pos` : 품사
* `meaning` : 한국어 뜻
* `example` : 학년/수능 수준 예문
* `level` : 학년 또는 수능 구분
* `difficulty` : B1 / B2 / C1
